什么是版本控制呢？

用来管理一个文件或者若干文件内容不断变化，不断更新，以便将来随时查阅以前的版本的系统。例如网页设计师需要保存某一页面布局中分所有修订版本，再例如毕业论文需要不断修改，等等。

git是什么？

git是目前最先进的版本控制系统，详细介绍

git操作：

  • 比较两个文件不相同的地方，用windows自带的工具也可以

    windows‘： fc 文件名 空格 文件名 EG:fc game_new.js game_old.js

    git :diff -u文件名 空格 文件名 EG:diff -u game_new.js game_old.js -u是标准区别格式，为了更好的阅读。

    • 查看修改的日志 git log，可以查看什么时候，谁，修改的什么东西。

Rss,新大陆！

发表于 2017-01-09 | 分类于 学习 |

事情发生在一个无聊的夜晚，我一边打理着自己的博客一边随便看一些无聊的东西，突然发现我的博客有个rss按钮，我的直接拿的别人的博客模板来做的，并不知道rss是个什么东西，，后来查阅后仿佛相见恨晚。

RSS是什么：Really Simple Syndication“聚合真的很简单”

我认为这句话足以解释所有。我现在想看不同类型的信息，不同平台，只要支持rss都可以在一个平台上看到，让你专注于内容，而不用担心是否是垃圾讯息，这样真的很方便，所以让自己的博客支持rss是一种美德。(●’◡’●)
最重要的是rss的广度几乎包含了你所需要的一切，只要你的信息源支持rss。我最喜欢的莫过于微信公众号，而且分门别类也是很棒的一点！
